# Remotion - videos para landing

Proyecto para generar videos MP4 que se publican en `landing/videos/`.

## Requisitos

- Node.js 20+
- npm

## Uso rapido

```bash
cd remotion
npm install
npm run build
```

Comandos individuales:

```bash
npm run render:dashboard
npm run render:clientes
```

Los renders se escriben directamente en:

- `landing/videos/demo-dashboard.mp4`
- `landing/videos/demo-clientes.mp4`

## Desarrollo visual

Para abrir Remotion Studio:

```bash
npm run dev
```
